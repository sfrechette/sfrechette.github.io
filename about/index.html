About

My name is Stéphane Fréchette, I’m a Cloud Solution Architect.
I have a passion for architecting, designing and building solutions that matter.

An all around data guy! Former SQL Server | Data Platform MVP, and Google Developer Expert - Google Cloud . Worked and fluent with several database platforms (SQL Server, MySQL, PostgreSQL, Neo4j, MongoDB and other NoSQL type databases) as well as programming languages such as (SQL, Bash, R, Python, JavaScript…)

  • Big Data; Hadoop, Spark, Streaming
  • ETL, ELT, building data pipelines

I’m quite passionate about reading, music and drumming. You can always count me in for some good food and fine wine. Born in Montréal, I currently live in Gatineau, Québec (Canada).
